# 명령어 정리

go run main.go는 기본적으로 실행해야 하기 떄문에 붙여주어야 한다.


# option

option으로는 다음과 같은 항목이 존재

`add -block <원하는 Data>`  : 블록을 추가 한다.

`print` : 방금 추가 된 블록을 가져 온다.

`printAll` : 모든 블록 데이터를 가져온다.
